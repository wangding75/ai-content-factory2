# Iteration 12 — 全局配置中心 V2

## 1. 目标

实现与已认可 UI 对齐的全局配置管理闭环，覆盖：

1. OpenAI-compatible LLM Provider 配置；
2. 通用工作流连接配置，当前连接类型仅支持 `n8n`；
3. 可供项目绑定的全局工作流配置；
4. 分发平台配置；
5. 类型目录、动态表单 Schema、密钥加密、乐观锁、幂等和审计。

本迭代只完成**配置数据新增、查询、修改与安全存储**，不访问任何真实第三方服务。

本迭代不提供硬删除接口。已保存配置通过后续迭代完成真实验证、启用和运行。

## 2. 核心边界

### 2.1 本迭代做什么

- 配置类型目录；
- 空状态、列表、搜索、筛选和添加/编辑抽屉；
- 配置创建、详情、列表和修改；
- API Key、Token 等凭证加密存储；
- 凭证编辑不回显；
- 连接类型创建后不可修改；
- 工作流必须绑定连接；
- 工作流类型由关联连接类型自动推导；
- 修改关联连接后，工作流类型和类型专属字段同步更新；
- 数据库迁移、约束、幂等、乐观锁和安全审计；
- UI 中保留验证、启用相关位置，但本迭代统一显示为“未接入”并禁用。

### 2.2 本迭代不做什么

- 不验证 LLM API Key；
- 不获取真实模型列表；
- 不调用真实 LLM API；
- 不实现 LLM Provider Adapter；
- 不验证 n8n 服务；
- 不调用 n8n API 或 Webhook；
- 不验证 Workflow ID / Webhook Path；
- 不实现 n8n Adapter；
- 不验证微信公众号、抖音、YouTube 或自定义平台凭证；
- 不实现 OAuth 流程；
- 不实现分发平台 Adapter；
- 不执行内容发布；
- 不创建 WorkflowRun；
- 不实现 callback server；
- 后端不得产生任何第三方出站网络请求。

## 3. 用户闭环

### 3.1 LLM 配置

空状态 → 添加配置 → 本地和服务端字段校验 → 保存 → 列表展示 → 查看详情 → 编辑。

规则：

- Provider 类型创建后不可修改；
- API Key 创建时可填写，编辑时不回显；
- 保存后状态固定显示“未接入”；
- “验证连接”“获取模型”和“启用”在本迭代禁用，并提示由后续迭代开放。

### 3.2 工作流连接

空状态 → 添加连接 → 选择连接类型 → 填写通用和类型专属字段 → 保存 → 列表展示 → 查看详情 → 编辑。

规则：

- 页面和 API 使用通用“连接”概念；
- 当前连接类型只有 `n8n`；
- 新增时连接类型可选择；
- 创建后连接类型不可修改；
- 凭证安全保存但不发起验证；
- 验证和启用控件在本迭代禁用。

### 3.3 工作流配置

空状态 → 添加工作流 → 选择关联连接 → 自动识别工作流类型 → 填写类型专属字段 → 保存 → 列表展示 → 查看详情 → 编辑。

规则：

- 每个工作流必须绑定一个连接；
- 工作流类型由连接类型推导，客户端不得独立提交或修改；
- 工作流允许修改关联连接；
- 关联连接改变后，服务器重新推导类型；
- 连接类型变化时，原类型专属字段必须清空或按新 Schema 重新校验；
- 本迭代不验证 Workflow ID 或 Webhook Path；
- 项目绑定工作流属于 Iteration 13。

### 3.4 分发平台配置

空状态 → 添加平台配置 → 选择平台类型 → 填写平台专属字段 → 保存 → 列表展示 → 查看详情 → 编辑。

规则：

- 平台类型创建后不可修改；
- 凭证加密保存且编辑不回显；
- 不验证平台凭证；
- 不执行真实发布；
- 验证和启用控件在本迭代禁用。

## 5. API

正式 OpenAPI 只覆盖：

- 四类配置的类型目录；
- 列表、创建、详情和修改；
- 搜索、筛选与分页；
- 凭证保留、替换和显式清除语义。

Iteration 12 不包含：

- `/verify`；
- `/enable`；
- `/disable`；
- LLM `/models`；
- 任何执行或发布接口。

冻结范围见 `api-scope.yaml`。正式开发时只修改仓库现有 OpenAPI 单一真值文件，不创建平行契约。

## 7. 页面壳层开发规则

1. 复用仓库现有共享侧边栏、顶部栏、页面标题区、面包屑和用户信息组件；
2. 只开发全局设置标题下方的工作区；
3. 不根据每张 Stitch 原型重新实现不同页面壳层；
4. 抽屉使用真实父列表页作为背景上下文；
5. 空状态和列表状态的添加按钮统一位于工作区标题栏右上角；
6. 空状态卡片内不实现重复添加按钮。
