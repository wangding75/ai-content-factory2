# Iteration 14 — WorkflowRun 运行时 — 验收标准（冻结）

## 1. UI 原型冻结

- [x] 新 Stitch 项目 R2 已生成；
- [x] P14_01 至 P14_10 人工 UI 验收通过；
- [x] 流程中心不包含“工作流配置”Tab；
- [x] 运行详情使用流程中心全局上下文；
- [x] 未绑定状态改为点击后的提示弹窗；
- [x] 项目概览补充运行摘要与快捷入口；
- [x] 项目侧不建设第二套运行记录页面。

## 2. 运行创建

- [ ] 四个环节均可从业务页面触发运行；
- [ ] 未绑定时不创建运行并展示 P14_09；
- [ ] 已绑定但依赖未就绪时不创建运行；
- [ ] 可运行时展示 P14_08；
- [ ] 创建 API 使用 `Idempotency-Key`；
- [ ] 相同 Key 与相同载荷重放不重复创建；
- [ ] 创建成功返回 202 并直接进入运行详情；
- [ ] 快照不含凭证或敏感 Header。

## 3. 状态机与 Worker

- [ ] queued 运行可被 Worker 领取并进入 running；
- [ ] 成功进入 succeeded；
- [ ] 执行、超时、Schema 或领域提交错误进入正确状态；
- [ ] queued/running 可按契约取消；
- [ ] 终态不可回退；
- [ ] 页面刷新后从数据库和事件恢复；
- [ ] 不依赖 callback server；
- [ ] n8n 输出通过 Webhook HTTP 响应返回。

## 4. 运行记录与筛选

- [ ] `/workflow-runs` 是唯一运行记录列表；
- [ ] 无工作流配置 Tab；
- [ ] 支持项目、环节、工作流、状态、时间和关键字筛选；
- [ ] 统计和表格使用相同筛选口径；
- [ ] 项目侧跳转自动带入 `projectId`；
- [ ] 环节入口同时带入 `stage`；
- [ ] 项目筛选可见、可清除；
- [ ] 空、加载、错误和成功状态正确区分；
- [ ] 列表失败不显示为空数据。

## 5. 运行详情

- [ ] 成功、运行中和失败复用统一详情结构；
- [ ] 面包屑为流程中心上下文；
- [ ] 所属项目是可点击属性，不是父级路由；
- [ ] 事件按 sequence 有序；
- [ ] 运行中可刷新，符合条件时可取消；
- [ ] 成功展示业务结果摘要和结果入口；
- [ ] 失败展示错误阶段、错误码、处理建议和折叠技术详情；
- [ ] 返回列表保留来源筛选；
- [ ] 不暴露凭证、Header、堆栈或完整上游响应。

## 6. 项目概览

- [ ] 项目运行摘要与 WorkflowRun 实际数据一致；
- [ ] 展示总运行、运行中、最近失败、最近运行；
- [ ] 展示最近三条运行；
- [ ] 查看全部和环节记录跳转正确；
- [ ] 项目侧没有重复的运行列表、筛选或详情实现；
- [ ] 无记录时展示轻量空态。

## 7. 重试与幂等

- [ ] 完整重试创建新运行并保留原运行；
- [ ] `retryOfRunId` 正确；
- [ ] 同一重试操作幂等；
- [ ] 不可重试状态返回标准错误；
- [ ] 领域提交重试不重新调用 n8n；
- [ ] 取消使用 expectedVersion；
- [ ] version 冲突返回 409 且不覆盖最新状态。

## 8. n8n Adapter 与配置状态

- [ ] Connection 可以真实验证；
- [ ] WorkflowConfiguration 可以真实验证；
- [ ] 只有满足条件的连接和工作流可启用；
- [ ] 配置变化后验证状态正确重置；
- [ ] 超时、HTTP 和格式错误归一化；
- [ ] 日志不含凭证、Authorization Header 或原始响应；
- [ ] 不实现其他 Adapter。

## 9. 数据与事务

- [ ] WorkflowRun、Event、快照和领域提交事务边界正确；
- [ ] Run Number 唯一；
- [ ] 每个运行事件 sequence 唯一；
- [ ] 项目摘要由运行表聚合，无冗余统计真相；
- [ ] 输出 Schema 非法时不写领域结果；
- [ ] 领域提交失败时不重复调用 n8n；
- [ ] 测试数据清理后数据库终态正确；
- [ ] 不要求验证历史数据库版本回滚。

## 10. 架构边界

- [ ] 不实现 callback server；
- [ ] 不直接调用 LLM Provider；
- [ ] 不获取 LLM 模型列表；
- [ ] 不验证、绑定或调用分发平台；
- [ ] 不执行 OAuth 或发布；
- [ ] 不建设项目级第二套运行记录；
- [ ] 不在流程中心增加工作流配置 Tab。

- [ ] 先局部验证 Adapter、状态机、筛选和摘要；

- [ ] 浏览器使用真实 API；
- [ ] Console 无未解释错误，Network 无未解释失败；
