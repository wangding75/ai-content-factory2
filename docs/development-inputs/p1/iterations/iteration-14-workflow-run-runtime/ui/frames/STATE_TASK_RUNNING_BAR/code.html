Component Spec

STATE_TASK_RUNNING_BAR

Reusable page-level asynchronous task status bar. Designed for top-of-canvas placement below the utility bar to provide non-blocking execution updates.

01. Queued status: "queued"
hourglass_empty
RUN-8F18 已进入队列 QUEUED

正在等待工作进程

02. Running status: "running"
sync
RUN-8F20 正在生成正文 RUNNING

正在生成并校验结构 · 已用时 1m12s

Padding: 16px (p-4) Radius: 12px (rounded-xl) Height: 72px (auto)
03. Succeeded status: "succeeded"
check_circle
RUN-8F24 重写已完成 SUCCESS

已创建 ContentVersion v5 候选

04. Failed status: "failed"
error
RUN-8F19 内容审核失败 FAILED

输出结构校验失败

info Usage Notes

  • Page navigation remains fully available and functional during active runs.
  • Status restores automatically after page refresh via persistent worker synchronization.
  • The bar serves as a progress summary and never exposes raw upstream responses or internal secrets.
