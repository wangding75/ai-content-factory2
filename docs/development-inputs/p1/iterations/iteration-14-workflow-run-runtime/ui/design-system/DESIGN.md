# AI Content Factory 2.0 商业化 SaaS 设计系统 R2

- 中文企业级 AI 内容生产后台，严格延续现有 Iteration 12/13 的成熟商业化风格。
- 固定左侧全局导航，包含概览、项目规划、内容生产、流程中心、项目设置等；流程中心是全局能力入口。
- 流程中心不设置“工作流配置”Tab，不设置任何 Tab，直接展示全局运行记录。
- 页面采用浅灰背景、白色内容面板、克制蓝色主操作、中性灰边框和分隔线。
- 状态颜色低饱和：成功绿色、运行中蓝色、失败红色、等待中灰色、取消中性灰。
- 禁止渐变、玻璃拟态、大面积高饱和色、夸张插画、厚重阴影、运维监控大屏风格。
- 组件圆角 8px，表格与筛选区信息密度适中，真实可开发。
- 运行详情页统一属于流程中心上下文，面包屑为“流程中心 / 运行记录 / 运行编号”，项目作为详情字段或可点击链接存在，不作为父级路由。
- 项目侧只展示运行摘要、最近运行和快捷入口，通过 projectId 与 stage 查询参数跳转到流程中心。
- 未绑定工作流场景使用点击“运行工作流”后弹出的提示弹窗，不使用完整独立页面。
- 全部文案为中文，桌面端 1440px 高保真商业化 SaaS 后台。
