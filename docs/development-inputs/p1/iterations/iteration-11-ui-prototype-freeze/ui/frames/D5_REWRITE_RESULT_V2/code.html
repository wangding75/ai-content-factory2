第9章 灰塔下的交易

罗杰裹紧了那件散发着霉味的灰斗篷，深吸一口略带海腥味的冷空气。眼前的灰塔在浓雾中若隐若现，像是一根刺破苍穹的枯指。他并不喜欢这个地方，尤其是今晚。塔门前原本应该有三名守卫，但此刻那里只剩下两团模糊的暗影，火把的微光在潮湿的空气里挣扎。

“你迟到了，罗杰。”一个如枯叶摩擦般的声音从阴影中传来。格里高利从黑暗中踱步而出，他的面容隐藏在宽大的兜帽下，只有那双混浊的眼睛闪烁着病态的光芒。“在这个时代，时间比黄金更脆弱。”

罗杰没有回答，只是冷冷地盯着对方。他能感觉到周围的空气正在变得粘稠，仿佛某种不可名状的存在正注视着这桩不洁的交易。这种令人窒息的克苏鲁式压迫感，他在圣地亚哥号沉没的那晚也曾感受过。他从怀中掏出一个用黑色丝绒包裹的长方形物体，递了过去。

“这是你要的名单，”罗杰的声音压得很低，带着一丝不容置疑的决绝，“但记住我们的约定，格里高利。如果名单上的任何一个名字在明天太阳升起前消失，这整座塔都会成为你的坟地。”

格里高利发出一阵令人毛骨悚然的低笑，枯瘦的手指急切地接过那个包裹。在触碰到包裹的一瞬间，他似乎颤抖了一下。雾气变得更浓了，将两人的身影彻底吞没在灰塔坚硬的石墙之下。
