第九章：灰塔下的交易

灰塔的影子像一柄巨大的黑色阔剑，斜斜地插在莫斯卡尔广场的废墟中。冷风穿过残破的拱门，发出的尖啸如同游魂的呜咽。阿瑟兰拉紧了兜帽，指尖轻轻触碰着腰间冰冷的匕首柄，皮革的纹理在寒风中显得格外粗糙。

“你迟到了，阿瑟兰。”一个低沉而沙哑的声音从塔基的阴影中传来。随着脚步声，一个披着灰色长袍的高大身影缓缓走入月光。他的脸上戴着一副由发黑的白银铸成的面具，面具上刻满了禁忌的符文，在微光下泛着幽幽的蓝光。

阿瑟兰冷笑一声，“在这样一个连耗子都不愿意出没的地方，准时并不是什么美德，尤里乌斯。”他停下脚步，与对方保持着五步的距离——这是最完美的爆发冲刺距离，也是逃脱暗算的防备红线。

尤里乌斯发出了像风干木头摩擦的声音，那大概是他的笑声。他从怀中掏出一个用黑色丝绸包裹的长方形物体，周围的空气似乎在那个物体出现的瞬间降了几度。“我要的东西呢？那颗属于旧神的‘余烬’。”

“在交付之前，我需要先看到你承诺的通行证，”阿瑟兰的眼神变得锐利，瞳孔中隐约有金色的流光闪过，“通往灰塔底层，接触那台‘起源引擎’的唯一凭证。”

广场周围的断壁残垣中，几双赤红的眼睛在暗处一闪而逝。阿瑟兰知道，这场交易远没有表面上看起来那么简单。灰塔不仅仅是一座废墟，它是这片大陆跳动的心脏，也是埋葬无数野心家的墓地。
