Developer Rules

Security & Privacy

  • PROHIBITED: Exposure of API keys or Bearer tokens.
  • PROHIBITED: Displaying raw n8n/LLM stack traces.
  • Always mask internal infrastructure URLs in error strings.

Functional Logic

  • Dismiss action (close) must only hide the UI node, not cancel the worker process.
  • Retry creates a new WorkflowRun record to preserve lineage.
  • Transaction-commit retry uses PUT /sync/recovery instead of a new run.
