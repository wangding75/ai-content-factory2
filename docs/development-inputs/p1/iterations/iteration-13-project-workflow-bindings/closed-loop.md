# Iteration 13 — 项目四环节工作流绑定 — Closed Loop

## 1. 入口

项目设置 → 工作流配置。

页面展示：

- 章节规划；
- 正文生成；
- 内容审核；
- 正文重写。

## 2. 绑定闭环

选择环节 → 打开绑定抽屉 → 搜索并选择全局工作流 → 查看关联连接和接入状态 → 填写默认参数 → 保存 → 返回总览。

保存结果：

- 绑定关系落库；
- 默认参数落库；
- 总览刷新；
- 不创建运行；
- 不访问第三方。

## 3. 未接入状态

当工作流或其关联连接为 `not_connected`：

- 允许保存绑定；
- 显示“已绑定，尚未接入执行能力”；
- 不显示为验证成功；
- 不提供本迭代可用的运行按钮；
- 后续 Iteration 14 完成真实验证和启用后，运行时再判断可执行性。

## 4. 异常闭环

- 未选择工作流：阻止保存；
- 工作流不存在或已归档：提示刷新候选；
- 参数 Schema 错误：定位字段并保留输入；
- 乐观锁冲突：提示绑定已被修改；
- 保存失败：抽屉保持打开；
- 页面刷新：从绑定 CRUD API 恢复；
- 不存在运行失败、日志和重试状态。

## 5. 与后续迭代关系

Iteration 14：

- 验证并启用 n8n 连接；
- 验证并启用工作流；
- 创建绑定快照；
- 创建并执行 WorkflowRun。
