# AI Content Factory 2.0｜领域术语表

| 术语 | 定义 |
|---|---|
| Project | 一次内容生产活动的核心上下文，P0 为小说项目 |
| ProjectPlanning | 项目的前提、受众、风格、目标和限制 |
| Material | 可跨项目复用的全局素材本体 |
| ProjectMaterialUsage | 某项目如何使用某个 Material 的关系 |
| PlotLine | 统一表示主线、子线及更深层故事线的树节点 |
| Foreshadowing | 可独立追踪的伏笔对象 |
| ChapterPlan | 章节候选或已确认规划 |
| ContentItem | 与一个生产单元对应的作品对象，P0 为章节 |
| ContentVersion | ContentItem 的不可覆盖正文快照 |
| ReviewReport | 对固定 ContentVersion 的审核结果 |
| ReviewFinding | 审核发现的问题 |
| ReviewRecommendation | 审核建议 |
| WorkflowRun | 一次生成、审核或重写执行记录 |
| Content Pack | 定义内容类型差异的扩展包，P0 为 Novel Pack |
| Workflow Provider | 执行生成、审核、重写能力的适配器，P0 为 Mock Provider |
| Current Version | ContentItem 当前默认展示版本 |
| Project Works | 当前项目的作品聚合读模型 |
| Global Works | 全部项目作品的全局聚合读模型 |
