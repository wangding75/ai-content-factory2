# AI Content Factory 2.0｜P0 指标与成功标准

P0 不以商业营收为验收目标，主要验证闭环、正确性和可扩展性。

## 1. 闭环指标

- 项目创建到重写版本的主链路可自动化执行。
- 所有 Iteration 02–08 核心需求均被 Iteration 09 E2E 覆盖。
- 关键失败分支验证无脏数据。

## 2. 数据正确性指标

- 重复素材绑定数量为 0。
- 审核覆盖正文事件数量为 0。
- 重写覆盖历史版本事件数量为 0。
- 无 WorkflowRun 的生成、审核或重写结果数量为 0。

## 3. 工程质量指标

- Go test、Web lint/typecheck、契约测试、数据库集成测试和 E2E 全部通过。
- 空库迁移成功率 100%。
- API 错误响应全部包含 request_id。
- 每个迭代验收后工作区干净并有唯一提交。

## 4. 产品质量指标

- 所有冻结页面具备真实路由和操作结果。
- 所有不可用能力具有明确禁用状态。
- 所有数据页面具有加载、空、错误和成功状态。
