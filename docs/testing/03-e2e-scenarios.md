# P0 E2E 场景

## E2E-01 项目创建

创建项目后进入概览，返回列表和刷新后仍可见。

## E2E-02 素材本体与用途

项目 A 创建人物，项目 B 绑定同一人物；编辑 B 用途不影响 A；解除 B 绑定后全局人物仍存在。

## E2E-03 故事线与伏笔

创建主线、子线和伏笔，刷新后树关系和计划章节保持正确。

## E2E-04 章节规划

Mock 生成候选，编辑、删除、确认；确认后不自动创建正文；显式进入生产后创建正文。

## E2E-05 审核不覆盖正文

保存 v1，执行审核，确认 v1 body 未变化且 ReviewReport 关联 v1。

## E2E-06 重写保留版本

从 v1 创建 v2，确认 v1 和 v2 均存在、parent 关系正确、current 仍按 P0 规则保持。

## E2E-07 全局聚合

全局素材和作品可查询到来源项目；流程中心显示对应 Mock WorkflowRun；设置显示真实能力状态。
