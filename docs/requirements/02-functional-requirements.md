# P0 功能需求目录

## FR-01 首页与项目

- FR-01-01：首页展示聚合统计和最近项目。
- FR-01-02：用户可查询项目列表。
- FR-01-03：用户可创建 Novel 项目。
- FR-01-04：创建后进入同一 projectId 的项目概览。
- FR-01-05：刷新和返回列表后项目仍存在。

## FR-02 项目策划

- FR-02-01：用户可读取项目策划。
- FR-02-02：用户可保存前提、受众、风格、目标和限制。

## FR-03 素材

- FR-03-01：创建全局素材。
- FR-03-02：项目内创建素材并绑定。
- FR-03-03：绑定已有素材。
- FR-03-04：编辑素材本体。
- FR-03-05：编辑项目用途。
- FR-03-06：解除项目绑定且保留素材。
- FR-03-07：查询素材引用项目。

## FR-04 叙事

- FR-04-01：创建主故事线。
- FR-04-02：创建子故事线。
- FR-04-03：展示故事线树。
- FR-04-04：创建和编辑伏笔。

## FR-05 章节规划

- FR-05-01：Mock 生成章节候选。
- FR-05-02：编辑未确认候选。
- FR-05-03：删除未确认候选。
- FR-05-04：批量确认候选。
- FR-05-05：已确认章节可显式进入正文生产。

## FR-06 正文与审核

- FR-06-01：为已确认章节创建 ContentItem。
- FR-06-02：Mock 生成或人工编辑正文。
- FR-06-03：保存正文草稿。
- FR-06-04：对固定版本执行 Mock 审核。
- FR-06-05：查询审核历史、问题和建议。

## FR-07 重写与作品

- FR-07-01：基于源版本和审核建议创建 Mock 重写。
- FR-07-02：重写创建新版本并保留父版本。
- FR-07-03：查询版本列表和详情。
- FR-07-04：查询项目作品。
- FR-07-05：查询全局作品。

## FR-08 能力与流程

- FR-08-01：查询内置 Mock 流程。
- FR-08-02：查询 WorkflowRun 状态。
- FR-08-03：查询能力和集成真实状态。
