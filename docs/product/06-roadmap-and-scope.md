# AI Content Factory 2.0｜范围与路线图

## P0：小说生产闭环

- Novel Pack。
- Mock Provider。
- 单用户。
- 项目、资产、故事线、章节、正文、审核、重写、版本。
- 全局素材、全局作品、流程和设置 Lite 页面。

## P1：真实能力与基础协作

候选方向：

- LLM Provider 配置与密钥管理。
- 真实章节规划、正文、审核和重写。
- n8n / Coze / ComfyUI Provider Adapter。
- 工作流模板配置和 Review Gate。
- 更完整版本比较和 current version 切换。
- 基础账号和工作区隔离。

## P2：平台化与多内容类型

候选方向：

- 短剧、视频脚本、图文等 Content Pack。
- 多人协作、角色权限和审核流。
- 发布适配器和发布记录。
- 计费、套餐、额度和用量。
- 工作流可视化编排。
- 多租户和规模化异步执行。

## P0 排除项

任何未在 P0 PRD、API 目录、数据模型目录和迭代计划中共同出现的能力，默认不进入 P0。不得因为 UI 中存在占位入口就自行实现。
